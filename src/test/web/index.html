大数据管理系统

gitlab地址：https://gitlab.suncaper.net/2018141461258/data-description

系统简介：本系统后台对购物数据进行处理，并显示相关信息，分析用户的购物习惯


系统功能概述：

个人信息查看：查看用户信息，且管理员能够使用搜索功能（右上角搜索图标）查看其他消费者的用户信息

群体画像查看：管理员能够查看用户的消费习惯等相关信息
